!
! Usage in Pi-hole (v5.0+ required for ABP syntax):
!   Option 1: Add as local adlist via file:///path/to/browsergames.txt
!   Option 2: Host via HTTP/HTTPS and add URL as adlist
!   After adding: run "pihole -g" (or Tools -> Update Gravity in web UI)
!
! ABP syntax explained:
!   ||domain.com^  =  blocks domain.com AND all subdomains (www., api., etc.)
!   Pi-hole 5.0+ auto-converts these into wildcard/regex block rules.
!
! For Pi-hole versions BELOW 5.0 that don't support ABP: use the .hosts
! variant of this file (if generated) or manually add regex rules like
! (\.|^)poki\.com$ under Domains -> Regex filter.
!
! ADDITIONAL MANUAL REGEX RULE (must be added separately in Pi-hole):
!   Regex:  brainrot

!            italian-brainrot-clicker.com, etc.)

!   (ABP adlist syntax cannot express substring wildcards - regex is required)
